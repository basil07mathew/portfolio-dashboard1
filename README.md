# Portfolio Dashboard

Professional portfolio tracking dashboard built with Next.js and Google Sheets.

## Features
- Real-time data from Google Sheets
- Auto-refresh every 5 minutes
- Professional dark theme
- Mobile responsive
- No backend needed

## Setup

1. Create Google Sheet with holdings data
2. Get Google Sheets API key
3. Update `pages/index.js` with your SHEET_ID and API_KEY
4. Deploy to Vercel

## Monthly Update
- Export holdings from Zerodha
- Update Google Sheet
- Dashboard auto-refreshes

## Deployment
- Deployed on Vercel: [vercel.com](https://vercel.com)
